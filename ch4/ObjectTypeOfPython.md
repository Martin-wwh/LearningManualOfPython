# 介绍python对象类型

1. 程序由模块构成
2. 模块包含语句
3. 语句包含表达式
4. 表达式创建并操作对象

## python的核心数据类型
内置对象
|  对象类型   | 实例  |
|  ----  | ----  |
| 数字  | 1234,3.1415 |
| 字符串  | 'spam' |
| 列表  | [1,[2,'three'],4] |
| 元组  | (1,2,'three',4) |
| 字典  | {1:2} |
| 文件  | file=open("test","r") |
| 集合  | set('abc'),{'a','b','c'} |
| 其他类型 | 类型/None/布尔型 |
| 编程单元类型 | 函数/模块/类 |
| 与实现相关的类型 | 编译的代码堆栈跟踪 |

python是动态类型语言，但同时它也是强类型语言(只能对一个对象进行适合该类型的有效操作)
